Régler le comportement du filtre

Le paramètre de filtre de Conradish détermine le mode initial lorsque vous ouvrez un document. Lorsque le filtrage est défini sur automatique, l'éditeur démarre en mode d'annotation, avec les indésirables suspects masqués. Lorsque le filtrage est défini sur manuel, l'éditeur démarre en mode de nettoyage, ce qui vous permet de corriger immédiatement les résultats de l'extension. Lorsque le filtrage est désactivé, l'éditeur démarre en mode d'annotation, avec les indésirables suspects visibles.

Selon les sites Web que vous visitez, l'un de ces paramètres s'avérera plus pratique que les autres. Lorsqu'un site fournit une feuille de style d'impression, le filtrage n'est généralement pas nécessaire. D'un autre côté, lorsqu'un site mêle de nombreuses publicités au texte principal de l'article, ou que la mise en page conduit à la sélection d'éléments de navigation, il est conseillé de le laisser activé.

Pour modifier le paramètre de filtre, cliquez sur le bouton d'extension de Chromed'Edge dans la barre d'outils, puis sur le bouton à trois points pour Conradish. Dans le menu, choisissez OptionsOptions d'extension :

La page d'options de Conradish s'ouvrira dans un nouvel onglet :

Autoriser les gros titres

Par défaut, Conradish réduit les titres H1 dans les articles à H2, une taille plus adaptée à la page imprimée. Si vous souhaitez les laisser tels quels, décochez la case Limiter la taille du titre.

Sélectionner des polices supplémentaires

Pour rendre des polices supplémentaires disponibles, cliquez sur Polices dans la barre latérale, puis sur le nom du script applicable (par exemple, arabe). Cochez la case à côté de la police que vous souhaitez pouvoir utiliser. Décochez la case si vous ne voulez pas qu'elle soit répertoriée. Vous pouvez trouver une police en utilisant le champ de recherche en haut de la page si vous connaissez le nom.

Imprimer un document

Pour imprimer, cliquez sur le bouton Imprimer. La fenêtre d'impression de Chromed'Edge apparaîtra. Vérifiez à nouveau que le Taille du papierFormat de papier correspond à ce qui est défini dans Conradish et que les Marges sont définies sur "Par défaut". Le document ne s'imprimera pas correctement si l'un de ces éléments est incorrect.
