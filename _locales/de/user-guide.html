Filterverhalten anpassen

Conradish's filter setting determines the initial mode when you open a document. When filtering is set to automatic, the editor starts in annotation mode, with suspected junks hidden. When filtering is set to manual, the editor starts in scrubbing mode, allowing you to immediately correct the extension's findings. When filtering is turned off, the editor starts in annotation mode, with suspected junks visible.

Depending on the web sites you visit, one of these settings will prove more convenient than the others. When a site provides a print stylesheet, filtering is generally not necessary. On the other hand, when a site intermingles a lot of ads and asides with the main article text, or the page layout leads to selection of navigational elements, then leaving it turned on is advisable.

To change the filter setting, click Chrome's extension button in the toolbar then the "kebab" button for Conradish. In the menu, choose Options:

Conradish's options page will open up in a new tab

Große Überschriften zulassen

By default, Conradish reduces H1 headings in articles to H2, a size more suitable for the printed page. If you wish to leave them as-is, uncheck the Limit heading size checkbox.

Auswählen zusätzlicher Schriftarten

Um zusätzliche Schriftarten verfügbar zu machen, klicken Sie in der Seitenleiste auf Schriftarten und dann auf den Namen der entsprechenden Schrift (z. B. Arabisch). Aktivieren Sie das Kontrollkästchen neben der Schriftart, die Sie verwenden möchten. Deaktivieren Sie das Kontrollkästchen, wenn Sie es nicht aufgelistet sehen möchten. Sie können eine Schriftart mithilfe des Suchfelds oben auf der Seite finden, wenn Sie den Namen kennen.

Drucken

Klicken Sie zum Drucken auf die Schaltfläche Drucken. Das Druckfenster von Chrome wird angezeigt. Vergewissern Sie sich, dass Papiergröße mit der Einstellung in Conradish übereinstimmt und dass Ränder auf "Standard" eingestellt ist. Das Dokument wird nicht korrekt gedruckt, wenn eine dieser Angaben falsch ist.

Um einen Artikel nach Microsoft Word zu kopieren, klicken Sie mit der rechten Maustaste auf das Dokument und wählen Sie „Alle auswählen“ (oder drücken Sie Strg-A):

Klicken Sie erneut mit der rechten Maustaste und wählen Sie "Kopieren" (oder drücken Sie Strg-C):

Erstellen Sie in Word ein neues Dokument und klicken Sie dann auf das Symbol „Einfügen“ in der Symbolleiste (oder drücken Sie Strg-V):

Der Text des Artikels wird nun zusammen mit den von Ihnen hinzugefügten Fußnoten in Word angezeigt:

Beachten Sie, dass die Online-Version von Word die angehängten Fußnoten derzeit nicht korrekt verarbeitet. Nur die Offline-Version von Microsoft Word funktioniert. WPS Office Writer funktioniert auch.
