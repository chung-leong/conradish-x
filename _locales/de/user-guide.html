Der übersetzte Satz erscheint dann in der Fußzeile der Seite:

Die Funktion Übersetzung hinzufügen unterscheidet sich von Definition hinzufügen nur dadurch, dass der Originaltext in der Fußnote weggelassen wird.

Sowohl Definition hinzufügen als auch Übersetzung hinzufügen werden im Menü angezeigt, wenn mehr als ein Wort (jedoch weniger als 8) ausgewählt ist.

Erklärung hinzufügen wird stattdessen im Menü angezeigt, wenn keine Übersetzung erfolgen würde (die Nach-Sprache ist die gleiche wie die Von-Sprache oder "Keine Sprache" ausgewählt).

Filterverhalten anpassen

Die Filtereinstellung von Conradish bestimmt den Anfangsmodus beim Öffnen eines Dokuments. Wenn die Filterung auf „Automatisch“ eingestellt ist, startet der Editor im Anmerkungsmodus, wobei mutmaßliche Junks ausgeblendet werden. Wenn die Filterung auf manuell eingestellt ist, startet der Editor im Scrubbing-Modus, sodass Sie die Ergebnisse der Erweiterung sofort korrigieren können. Wenn die Filterung deaktiviert ist, startet der Editor im Anmerkungsmodus, wobei mutmaßliche Junks sichtbar sind.

Je nachdem, welche Websites Sie besuchen, erweist sich eine dieser Einstellungen als praktischer als die anderen. Wenn eine Website ein Druck-Stylesheet bereitstellt, ist eine Filterung im Allgemeinen nicht erforderlich. Wenn andererseits eine Website viele Anzeigen mit dem Hauptartikeltext vermischt oder das Seitenlayout zur Auswahl von Navigationselementen führt, ist es ratsam, sie eingeschaltet zu lassen.

Um die Filtereinstellung zu ändern, klicken Sie in der Symbolleiste auf die Erweiterungsschaltfläche von Chrome und dann auf die Schaltfläche mit den drei Punkten für Conradish. Wählen Sie im Menü Optionen:

Große Überschriften zulassen

Standardmäßig reduziert Conradish H1-Überschriften in Artikeln auf H2, eine Größe, die besser für die gedruckte Seite geeignet ist. Wenn Sie sie unverändert lassen möchten, deaktivieren Sie das Kontrollkästchen Überschriftengröße begrenzen.

Zusätzlicher Schriftarten auswählen

Um zusätzliche Schriftarten verfügbar zu machen, klicken Sie in der Seitenleiste auf Schriftarten und dann auf den Namen der entsprechenden Schrift (z. B. Arabisch). Aktivieren Sie das Kontrollkästchen neben der Schriftart, die Sie verwenden möchten. Deaktivieren Sie das Kontrollkästchen, wenn Sie es nicht aufgelistet sehen möchten. Sie können eine Schriftart mithilfe des Suchfelds oben auf der Seite finden, wenn Sie den Namen kennen.

Dokument drucken

Klicken Sie zum Drucken auf die Schaltfläche Drucken. Das Druckfenster von Chrome wird angezeigt. Vergewissern Sie sich, dass Papierformat mit der Einstellung in Conradish übereinstimmt und dass Ränder auf "Standard" eingestellt ist. Das Dokument wird nicht korrekt gedruckt, wenn eine dieser Angaben falsch ist.

Um einen Artikel nach Microsoft Word zu kopieren, klicken Sie mit der rechten Maustaste auf das Dokument und wählen Sie „Alle auswählen“ (oder drücken Sie Strg-A):

Klicken Sie erneut mit der rechten Maustaste und wählen Sie "Kopieren" (oder drücken Sie Strg-C):

Erstellen Sie in Word ein neues Dokument und klicken Sie dann auf das Symbol „Einfügen“ in der Symbolleiste (oder drücken Sie Strg-V):

Der Text des Artikels wird nun zusammen mit den von Ihnen hinzugefügten Fußnoten in Word angezeigt:

Beachten Sie, dass die Online-Version von Word die angehängten Fußnoten derzeit nicht korrekt verarbeitet. Nur die Offline-Version von Microsoft Word funktioniert. WPS Office Writer funktioniert auch.
